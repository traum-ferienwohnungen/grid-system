TFG Grid System

TFW grid system uses a series of containers, rows, and columns to layout and align content. It’s built with flexbox and is fully responsive. Below is an example and an in-depth look at how the grid comes together.

<div class="grid"> <div class="grid__row"> <div class="grid__column grid__column--4">grid__column</div> <div class="grid__column grid__column--4">grid__column</div> <div class="grid__column grid__column--4">grid__column</div> </div> </div>

The above example creates three equal-width columns on medium and large devices using our predefined grid classes. Those columns are centered in the page with the parent .grid.

Breaking it down, here’s how it works

  • Containers provide a means to center and horizontally pad your site’s contents. Use .grid for a responsive width in percentages or .grid--full for width: 100% across all viewport and device sizes.
  • Rows are wrappers for columns. Each column has horizontal padding (called a gutter) for controlling the space between them. This padding is then counteracted on the rows with negative margins. This way, all the content in your columns is visually aligned down the left side.
  • In a grid layout, content must be placed within columns and only columns may be immediate children of rows.
  • Column classes indicate the number of columns you’d like to use out of the possible 12 per row. So, if you want three equal-width columns across, you can use .grid__column .grid__column--4.
  • Column widths are set in percentages, so they’re always fluid and sized relative to their parent element.

Paddingless Columns

Sometimes it is necessary to create a paddingless columns, especially on mobile devices, for seamless slider from end to end of the screen for example. You can easily add .grid__column--paddingless to each column if you want to set that specific column to no padding.

<div class="grid"> <div class="grid__row"> <div class="grid__column grid__column--paddingless">...</div> <div class="grid__column grid__column--m-paddingless">...</div> <div class="grid__column grid__column--s-paddingless">...</div> </div> </div>

Spaced Grid

It is often the case that you want to create an evenly distributed grid when using multiple .grid_rows. To achieve that kind of distribution just add the modifier --spaced-evenly to your parent grid. Every column will get a spacing set on the bottom defined by $grid-gutter-width (default: 20px) except the last one. This is especially useful in boxed layouts.

Row Directions

It is possible to manipulate the flex direction of a row by adding certain modifiers to the .grid__row. Modifiers are:
--{size}-direction-r Direction: row
--{size}-direction-rr Direction: row-reverse
--{size}-direction-c Direction: column
--{size}-direction-cr Direction: column-reverse
